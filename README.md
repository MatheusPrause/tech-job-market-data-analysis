# Tech Job Market Data Analysis (Brazil)

## Objective
This project aims to analyze the Brazilian technology job market using real-world data to identify trends related to salaries, required skills, seniority levels, and geographic distribution.

The project is developed progressively throughout my Data Analysis career path, with continuous improvements as new tools and concepts are learned.

---

## Key Questions
- Which technology roles offer the highest average salaries in Brazil?
- How do salaries vary across seniority levels?
- What technologies and skills are most frequently required?
- Are there significant salary differences between regions?
- How does remote work influence compensation?
